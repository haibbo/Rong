# S3 — Object Storage

S3-compatible object storage client. Works with AWS S3, Cloudflare R2, DigitalOcean Spaces, and any S3-compatible service.

## Quick Start

```javascript
const client = new S3Client({
  accessKeyId: "your-access-key",
  secretAccessKey: "your-secret-key",
  bucket: "my-bucket",
  endpoint: "https://s3.us-east-1.amazonaws.com",
});

// Write
await client.write("hello.txt", "Hello World!");

// Read
const file = client.file("hello.txt");
const text = await file.text();

// Delete
await client.delete("hello.txt");
```

## Credentials

Credentials are resolved from environment variables, then overridden by options.

| Option | Primary Env Var | Fallback Env Var |
|--------|----------------|-----------------|
| `accessKeyId` | `S3_ACCESS_KEY_ID` | `AWS_ACCESS_KEY_ID` |
| `secretAccessKey` | `S3_SECRET_ACCESS_KEY` | `AWS_SECRET_ACCESS_KEY` |
| `bucket` | `S3_BUCKET` | `AWS_BUCKET` |
| `region` | `S3_REGION` | `AWS_REGION` |
| `endpoint` | `S3_ENDPOINT` | `AWS_ENDPOINT` |
| `sessionToken` | `S3_SESSION_TOKEN` | `AWS_SESSION_TOKEN` |

Default `Rong.s3` singleton reads env vars automatically:

```javascript
// Uses env vars — no config needed
const file = Rong.s3.file("my-key.txt");
```

## S3Client

### Constructor

```javascript
const client = new S3Client({
  accessKeyId: "...",
  secretAccessKey: "...",
  bucket: "my-bucket",
  region: "us-east-1",            // default: "us-east-1"
  endpoint: "https://...",         // custom endpoint for R2/Spaces/etc.
  sessionToken: "...",             // optional STS token
  acl: "public-read",             // optional default ACL
  virtualHostedStyle: true,        // default: false (path style)
});
```

### Methods

#### `client.file(path, options?)`

Returns a lazy `S3File` reference. **No network request** — synchronous.

```javascript
const file = client.file("data/report.json");
const file2 = client.file("other.txt", { bucket: "other-bucket" });
```

#### `await client.write(path, data, options?)`

Upload data to S3. Returns the number of bytes written.

```javascript
await client.write("file.txt", "Hello World!");
await client.write("data.json", JSON.stringify(obj), {
  type: "application/json",
});
```

**Accepted data types:** `string`, `ArrayBuffer`, `Uint8Array`

#### `await client.delete(path)` / `await client.unlink(path)`

Delete an object.

```javascript
await client.delete("old-file.txt");
```

#### `await client.exists(path)`

Check if an object exists. Returns `boolean`.

```javascript
if (await client.exists("config.json")) {
  // ...
}
```

#### `await client.size(path)`

Get the object size in bytes.

```javascript
const bytes = await client.size("large-file.bin");
```

#### `await client.stat(path)`

Get object metadata.

```javascript
const stat = await client.stat("file.txt");
// {
//   etag: "\"abc123...\"",
//   lastModified: "2025-01-15T10:30:00Z",
//   size: 1024,
//   type: "text/plain"
// }
```

#### `await client.presign(path, options?)`

Generate a presigned URL.

```javascript
const getUrl = await client.presign("file.txt");
const putUrl = await client.presign("upload.bin", {
  expiresIn: 3600,     // seconds, default: 86400 (24h)
  method: "PUT",       // "GET" | "PUT" | "DELETE"
});
```

#### `await client.list(options?)`

List objects in the bucket.

```javascript
const result = await client.list({
  prefix: "uploads/",
  maxKeys: 100,
  startAfter: "uploads/last-key",  // pagination
});

// result.contents = [{ key, size, lastModified, etag }, ...]
// result.isTruncated = boolean

// Pagination
if (result.isTruncated) {
  const lastKey = result.contents.at(-1).key;
  const next = await client.list({
    prefix: "uploads/",
    maxKeys: 100,
    startAfter: lastKey,
  });
}
```

## S3File

Lazy reference to an S3 object. Created via `client.file()`. No network request until you call an async method.

### Reading

```javascript
const file = client.file("data.json");

const text = await file.text();
const data = await file.json();
const buffer = await file.bytes();        // ArrayBuffer
const ab = await file.arrayBuffer();      // alias
```

### Writing

```javascript
await file.write("new content");
await file.write(JSON.stringify(data), { type: "application/json" });
```

### Partial Reads

```javascript
const first1k = file.slice(0, 1024);
const header = await first1k.text();
```

### Metadata

```javascript
file.name;                    // "data.json"
file.size;                    // NaN (use stat() instead)
await file.exists();          // boolean
await file.stat();            // { etag, lastModified, size, type }
```

### Delete

```javascript
await file.delete();
await file.unlink();          // alias
```

### Presigned URL

```javascript
const url = await file.presign({ expiresIn: 3600 });
const putUrl = await file.presign({ method: "PUT" });
```

## Compatible Services

| Service | Endpoint |
|---------|----------|
| AWS S3 | `https://s3.<region>.amazonaws.com` |
| Cloudflare R2 | `https://<account-id>.r2.cloudflarestorage.com` |
| DigitalOcean Spaces | `https://<region>.digitaloceanspaces.com` |
| Google Cloud Storage | `https://storage.googleapis.com` |
| Supabase Storage | `https://<id>.supabase.co/storage/v1/s3/storage` |
| rustfs (dev/test) | `http://localhost:9000` |

## Error Handling

```javascript
try {
  await client.write("file.txt", data);
} catch (e) {
  // e.name === "S3Error"
  // e.message contains details
}
```

Error codes:
- `ERR_S3_MISSING_CREDENTIALS` — credentials or bucket not configured
- `ERR_S3_INVALID_METHOD` — invalid presign method
- `ERR_S3` — general S3 operation error
