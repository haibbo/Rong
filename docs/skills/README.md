# Rong Agent Skills

This directory is the source of truth for Rong agent skills.

The `packages/skill/` package turns these docs into installable, self-contained skill
folders. Do not hand-edit `packages/skill/assets/`; it is generated by
`node packages/skill/bin/pack.mjs`.

## Skills

- `rong-runtime-developer`: write JavaScript for the Rong runtime. Its packaged
  API references are generated from `docs/api/*.md`.
- `rong-module-author`: write Rust modules that expose functions, classes,
  type conversions, and errors to JavaScript.

## Validate

```bash
npm --prefix packages/skill run check
```
