# Rong JavaScriptCore Backend

This crate provides the JavaScriptCore (JSC) backend for RongJS.

- Crate: `rong_jscore`
- Purpose: Integrates WebKit's JavaScriptCore engine with RongJS
- Usage: Enable the `jscore` feature on `rong`
